Generate Legacy Bitcoin Address
===============================

C++ Implementation
------------------
The `int SHA256_Final(unsigned char *md, SHA256_CTX *c)` function places the message digest in `md`, which must have space for `SHA256_DIGEST_LENGTH` bytes (i.e. 32) bytes.
